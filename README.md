# Password Manager (Django)

Project description:
A password manager is a web application built using Django that aims to provide a secure and reliable way to store and manage users ' passwords. It allows users to add, review, modify their passwords, while ensuring high protection using modern encryption technologies.

## Features:

- Add and save passwords: allows users to add their own passwords and store them securely.

- Data encryption: passwords are encrypted using powerful algorithms to protect them from any hacking attempt.

- Easy management: simple and easy-to-use user interface for reviewing and managing stored passwords.

- Multiple security levels: includes the application of restrictions for access and protection of accounts using two-factor authentication (2FA) when needed.

## Techniques used:

- Django (Back-End)

- SQLite (database)

- HTML/CSS (for interface design)

## How to operate:

1- Upload or clone this repository.

2- Install the requirements using
``` bash
pip install -r requirements.txt
```

3- Set up the database and run the server using python manage.py migrate then python manage.py runserver.

Now you can access the application via the browser.


***Note:*** be sure to follow security best practices when using the app in a production environment.
